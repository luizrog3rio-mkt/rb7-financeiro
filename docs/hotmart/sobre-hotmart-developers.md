# Sobre Hotmart Developers

> Referência: documentação Hotmart Developers — "Sobre Hotmart Developers".

O objetivo desta API é disponibilizar os serviços da Hotmart na forma de produtos de APIs — ajudando a resolver problemas do negócio, escalar com facilidade e focar no que importa.

## Introdução

Com a API você tem acesso a dados importantes do negócio: informações dos clientes, como eles consomem o conteúdo, e várias outras características relevantes.

### Características técnicas

- **Padrão REST:** a API segue o padrão `REST` para consumo simples e claro.
- **URLs orientadas a recursos:** todas as operações usam URLs com nomenclatura de fácil entendimento.
- **Verbos HTTP:** a comunicação cliente ↔ serviço usa `GET`, `POST`, `PUT`, `PATCH` e `DELETE`.
- **Respostas em JSON:** os retornos vêm no formato `JSON`.

### Sandbox

Há um ambiente de **[sandbox](https://developers.hotmart.com/docs/pt-BR/start/sandbox)** para testar a integração durante o desenvolvimento. Ele:

- Possui **todos os endpoints** da API e simula o ambiente de produção.
- É **completamente isolado** da produção — você não precisa se preocupar em modificar recursos reais na Hotmart.
- Usa apenas **dados fictícios**.
