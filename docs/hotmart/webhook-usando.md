# Webhooks — Usando Webhooks

> Referência: documentação Hotmart Developers — seção "Webhook" > "Usando Webhooks".

Os webhooks notificam a sua estrutura sempre que um evento acontece nos seus produtos (pagamento confirmado, contestação de cobrança, pagamento recorrente bem-sucedido, entre outros). O histórico de eventos fica armazenado por até **60 dias**.

## Passos para integrar via webhooks

1. **Criar um endpoint** ou local para receber as informações.
2. **Configurar** qual é o produto, os eventos e o endereço de recebimento.
3. **Testar** a integração.

> A configuração e o teste são feitos no painel da Hotmart ("Criar uma configuração").
